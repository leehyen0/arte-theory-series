# Series Map

## Paper 1: Bias as Controlled Asymmetry

**Role:** selection.

Bias is framed as controlled asymmetry: a mechanism that gives differential weight to candidate interpretations, actions, or hypotheses.

Repository: https://github.com/leehyen0/bias-as-controlled-asymmetry

## Paper 2: Complementary Possibility Axes

**Role:** preservation.

Complementary axes preserve unrealized alternatives that selection would otherwise discard.

Repository: https://github.com/leehyen0/complementary-possibility-axes

## Paper 3: Temporal Superposition and Self-Transcendent Metacognition

**Role:** temporal integration.

Temporal superposition integrates:

- past evidence,
- present execution,
- preserved alternatives,
- possible future states.

Self-transcendent metacognition is used in a computational-only sense: present-policy revision by a wider temporal self-model. It is not a metaphysical claim.

Repository: https://github.com/leehyen0/temporal-superposition-self-transcendence

## Paper 4: QuestionField Theory

**Role:** inquiry and perturbation.

QuestionField treats a question as a generative perturbation that induces:

- candidate directions,
- ambiguity preservation,
- operator pressure,
- verification burden,
- next-seed proposals.

Repository: https://github.com/leehyen0/questionfield-theory

## Paper 5: LawGenesis Theory

**Role:** candidate-law testing.

LawGenesis tests candidate patterns through:

- counterexample pressure,
- verification gates,
- consumption traces,
- revision or rejection.

Repository: https://github.com/leehyen0/lawgenesis-theory

## Unified chain

```text
Bias
→ Complementary Preservation
→ Temporal Integration
→ Question-Induced Search
→ Candidate-Law Testing
```

or:

```text
select
→ preserve unrealized alternatives
→ integrate past evidence, present execution, preserved alternatives, and possible future states
→ determine what should be actively asked next
→ test candidate patterns through counterexample pressure, verification gates, and later consumption traces
```
