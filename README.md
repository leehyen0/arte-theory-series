# Arte Theory Series

A theory-first public index for a five-paper sequence on selection, preserved alternatives, temporal integration, question-induced search, and candidate-law formation.

This repository is an **index and reading map**, not a proof repository. It links the individual public drafts and explains how they connect.

## One-line chain

> Bias selects; complementary axes preserve unrealized alternatives; temporal superposition integrates past evidence, present execution, preserved alternatives, and possible future states; QuestionField determines what should be actively asked next; LawGenesis tests candidate patterns through counterexample pressure, verification gates, and later consumption traces.

## What this series is

This series is a set of public theory-first drafts. It proposes a conceptual and computational sequence:

1. selection,
2. preservation of unrealized alternatives,
3. temporal integration,
4. question-induced search,
5. candidate-law testing and later consumption.

The intended audience is readers interested in cognitive architecture, computational creativity, counterfactual reasoning, active inquiry, agent design, and theory-building workflows.

## What this series does **not** claim

This series does **not** claim:

- AGI or superintelligence,
- consciousness or sentience,
- paranormal precognition,
- literal quantum cognition or literal quantum consciousness,
- model-weight modification,
- public/offbox proof,
- official benchmark validation,
- validated experimental results,
- externally reproduced performance improvement,
- safety certification or deployment readiness.

Internal development traces, local logs, prompt histories, and private runtime artifacts are development context only. They are not public proof, external reproduction, benchmark validation, or evidence of model-weight modification.

## Recommended reading order

Start with Paper 1 and Paper 2. They define the selection/preservation pair.

Then read Paper 3 for temporal integration and future-state metacognition.

Then read Paper 4 for questions as operators that perturb search and generate verification pressure.

Finally read Paper 5 for candidate-law formation, counterexample pressure, and consumption traces.
